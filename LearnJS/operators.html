операторы сравнения

ОПЕРАТОР ОПИСАНИЕ
== Оператор "равно".
если 45 == 45 то результат будет TRUE
!= Оператор "неравно"
 если 45 != 3 то тогда будет значение TRUE  
=== Оператор "строго равно"
 если 45 === "45" то тогда будет значение FALSE
 этот оператор сравнивает не только значение но и тип
!== Оператор "строго неравно"
 если 45 !== "45" то тогда будет значение TRUE
 этот оператор сравнивает не только значение но и тип
> Оператор "больше"
 если 45 > 40 то тогда будет значение TRUE
>= Оператор "больше или равно"
 если 45 >= 45 то тогда будет значение TRUE
Оператор "меньше"
 если 35 меньше 45 то тогда будет значение TRUE
Оператор "меньше или равно"
 если 45  меньше или равен 45 то тогда будет значение TRUE

Логические операторы

логические операторы используют
при работе с логическими значениями TRUE  и FALSE .
Операторов в JS всего три.

Оператор Описание
&& Логическое И:
Это бинарный оператор,результатом выражения A && B будет  
TRUE если оба операнда равны
здесь все значения равны и вывод TRUE
здесь разные значения и вывод  FALSE
|| Логическое ИЛИ:
Это бинарный оператор,результатом выражения A || B
будет TRUE если если хотябы один операнд равен TRUE
если оба операнда равны FALSE   то результат выражения
тоже будет FALSE
! Оператор логического отрицания:
это унарный оператор.
Если значение операнда TRUE ,то результата выражения
!TRUE будет FALSE .
  

Таблица истинности для логического И(&&)

A B TRUE FALSE
TRUE TRUE FALSE
FALSE FALSE FALSE

Таблица истинности для логического ИЛИ(||)

A B TRUE FALSE
TRUE TRUE TRUE
FALSE TRUE FALSE

Побитовые операторы

Имеется группа операций,которые выполняются с целыми числами на уровне их битового представления.
Такие операции называются побитовыми или поразрядными, как и их операторы.
Для работы с такими операциями нужно иметь некоторое представление о двоичной системе счисления и способах битового представления чисел в компьютере.

Детали

В двоичной системе счисления числа записываются с помощью всего двух цифр - нулей и единиц.
Например, двоичный код 101 в десятичной равен 5 , а код 10101 это 21 в десятичной.

основная сфера применения масок – это быстрые вычисления, экономия памяти, низкоуровневые операции, связанные с рисованием из JavaScript (3d-графика), интеграция с некоторыми функциями ОС (для серверного JavaScript), и другие ситуации, когда уже существуют функции, требующие битовую маску.

Правила бинарного сложения чисел

0 + 0 = 0
0 + 1 = 1
1 + 0 = 0
1 + 1 = 10

Сокращенные формы оператора присваивания

В   JavaScript есть специальные формы сокращения операторов присваивания.
Допустим,необходимо выполнить команду вида A = A + B ,
тогда эту же команду можно записать в более компактном виде , A+=b

Оператор Пример использования Эквивалентная команда
+= A+=B A=A+B
-= A-=B A=A-B
*= A*=B A=A*B
/= A/=B A=A/B
%= A%=B A=A%B
&= A&=B A=A&B
|= A|=B A=A|B
^= A^=B A=A^B

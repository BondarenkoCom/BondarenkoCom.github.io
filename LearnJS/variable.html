знакомство и работа с переменными

Переменная-это именнованная область памяти ,к которой можно обращаться по имени
таким образом у переменной есть имя (которое дал разработчик)
 так же у переменной есть тип

значения которые использует JavaScript

  • String - текстовая строка;
  • number - числовое значение;
  • boolean - логическое (истина или ложь);
  • object- объект;
  • function - функция;

сценарий с одной переменной

var a;
a = "вывод этого сообщения";
alert(a);

Сценарий с двумя переменными

var a,b;
a ="мой возраст";
b = 25;
document.write(a + b);

видно что команда document.write() вывела и произвела склеивание(конкатенацию) обеих значений из одной и той же переменной

работа с выражениями

существует функция eval()
она меняет текстовые типы на математические
Пример:
выражение "3+(5*2+6)/4" это string ,хотя очевидно что здесь арифмитическое выражение (number)

var x = "3+(5*2+6)/4";
document.write(x +"="); // эта команда добавит знак равентсва
document.write(eval(x));

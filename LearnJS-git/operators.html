основные операторы

 есть четыре вида операторов:

  • Арифметические операторы
  • Операторы сравнения
  • Логические операторы
  • Побитовые операторы

арифметические операторы

ОПЕРАТОР ОПИСАНИЕ
+ Оператор сложения.
(20+32)
- Оператор вычитания
(120-32)
* Оператор умножения
(20*32)
/ Оператор деления
(120/20)
% Оператор деления с остатком
(17%3=5(ост 2))
здесь мы увидим только остаток
++ Оператор инкремента
(++5)
-- Оператор декремента
(--5)

операторы сравнения

ОПЕРАТОР ОПИСАНИЕ
== Оператор "равно".
если 45 == 45 то результат будет TRUE
!= Оператор "неравно"
 если 45 != 3 то тогда будет значение TRUE  
=== Оператор "строго равно"
 если 45 === "45" то тогда будет значение FALSE
 этот оператор сравнивает не только значение но и тип
!== Оператор "строго неравно"
 если 45 !== "45" то тогда будет значение TRUE
 этот оператор сравнивает не только значение но и тип
> Оператор "больше"
 если 45 > 40 то тогда будет значение TRUE
>= Оператор "больше или равно"
 если 45 >= 45 то тогда будет значение TRUE
Оператор "меньше"
 если 35 меньше 45 то тогда будет значение TRUE
Оператор "меньше или равно"
 если 45  меньше или равен 45 то тогда будет значение TRUE

Логические операторы

логические операторы используют
при работе с логическими значениями TRUE  и FALSE .
Операторов в JS всего три.

Оператор Описание
&& Логическое И:
Это бинарный оператор,результатом выражения A && B будет  
TRUE если оба операнда равны
здесь все значения равны и вывод TRUE
здесь разные значения и вывод  FALSE
|| Логическое ИЛИ:
Это бинарный оператор,результатом выражения A || B
будет TRUE если если хотябы один операнд равен TRUE
если оба операнда равны FALSE   то результат выражения
тоже будет FALSE
